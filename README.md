# 🎓 SUZA Alumni Management System

> A full-stack web platform connecting graduates of the **State University of Zanzibar (SUZA)**.  
> Built as a Final Year Project (FYP) using Python FastAPI and React.

---

## 📋 Project Overview

The **SUZA Alumni Management System** is a web-based platform developed to bridge the gap between the State University of Zanzibar and its graduates. It enables alumni to register, build professional profiles, access job opportunities, participate in alumni clubs, and stay informed about university events.

The platform includes a powerful administrative dashboard for managing alumni data, events, job postings, announcements, and certificate requests — all in one place.

---

## ✨ Features

### 👩‍🎓 Alumni
- Register and manage a professional profile
- Browse the alumni directory with search and filter
- View and RSVP to upcoming events
- Join and leave alumni clubs/communities
- Access job listings posted within the network
- Send and receive direct messages with other alumni
- View university announcements on the bulletin board
- Request certificate reissuance or verification
- Add personal achievements to profile

### 🔧 Admin
- Full admin dashboard with platform statistics
- Manage alumni accounts (view, deactivate)
- Post and remove job listings
- Create and manage events
- Create and manage clubs
- Post announcements
- Approve or reject certificate requests
- Seed data auto-loaded on first startup

## 🚀 Installation & Setup

### Prerequisites
Make sure you have the following installed:
- [Python 3.10+](https://www.python.org/downloads/)
- [Node.js (LTS)](https://nodejs.org/)
- [Git](https://git-scm.com/)

---

### 1. Clone the Repository

```bash
git clone https://github.com/Mahembajr/alumni_app.git
cd alumni_app
```

---

### 2. Backend Setup

```bash
cd backend
```

Install dependencies:

```bash
pip install fastapi "uvicorn[standard]" "python-jose[cryptography]" "passlib[bcrypt]" python-multipart sqlalchemy "pydantic[email]" "bcrypt==4.0.1"
```

Start the backend server:

```bash
python -m uvicorn main:app --reload --port 8000
```

The API will be running at: **http://127.0.0.1:8000**  
Interactive API docs available at: **http://127.0.0.1:8000/docs**

---

### 3. Frontend Setup

Open a **new terminal window** and run:

```bash
cd frontend
npm install
npm run dev
```

The app will be running at: **http://localhost:5173**

---

### 4. Default Admin Login

```
Email:    admin@suza.ac.tz
Password: admin123
```

> ⚠️ Change the admin password and the JWT secret key before any production deployment.

## 📡 API Documentation

The full interactive API documentation is auto-generated by FastAPI and available at:

```
http://127.0.0.1:8000/docs
```

### Key Endpoints

#### 🔐 Authentication
| Method | Endpoint | Description |
|---|---|---|
| POST | `/api/auth/register` | Register a new alumni |
| POST | `/api/auth/login` | Login and receive JWT token |
| GET | `/api/auth/me` | Get current logged-in user |

#### 👤 Users
| Method | Endpoint | Description |
|---|---|---|
| GET | `/api/users` | List all alumni (with search/filter) |
| GET | `/api/users/{id}` | Get a specific user |
| PUT | `/api/users/me` | Update own profile |
| DELETE | `/api/users/{id}` | Deactivate a user (Admin only) |

#### 💼 Jobs
| Method | Endpoint | Description |
|---|---|---|
| GET | `/api/jobs` | List all active jobs |
| POST | `/api/jobs` | Post a new job |
| DELETE | `/api/jobs/{id}` | Remove a job (Admin only) |

#### 📅 Events
| Method | Endpoint | Description |
|---|---|---|
| GET | `/api/events` | List all events |
| POST | `/api/events` | Create an event (Admin only) |
| POST | `/api/events/{id}/rsvp` | Toggle RSVP for an event |
| DELETE | `/api/events/{id}` | Delete an event (Admin only) |

#### 👥 Clubs
| Method | Endpoint | Description |
|---|---|---|
| GET | `/api/clubs` | List all clubs |
| POST | `/api/clubs` | Create a club (Admin only) |
| POST | `/api/clubs/{id}/join` | Join or leave a club |
| DELETE | `/api/clubs/{id}` | Delete a club (Admin only) |

#### 📢 Announcements
| Method | Endpoint | Description |
|---|---|---|
| GET | `/api/announcements` | List all announcements |
| POST | `/api/announcements` | Post an announcement (Admin only) |
| DELETE | `/api/announcements/{id}` | Delete announcement (Admin only) |

#### ✉️ Messages
| Method | Endpoint | Description |
|---|---|---|
| GET | `/api/messages` | Get all messages for current user |
| POST | `/api/messages` | Send a message |

#### 📄 Certificates
| Method | Endpoint | Description |
|---|---|---|
| GET | `/api/certificates` | List certificate requests |
| POST | `/api/certificates` | Submit a certificate request |
| PUT | `/api/certificates/{id}` | Approve/Reject request (Admin only) |

#### 📊 Admin Stats
| Method | Endpoint | Description |
|---|---|---|
| GET | `/api/admin/stats` | Get platform statistics (Admin only) |
